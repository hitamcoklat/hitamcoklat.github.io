## 1. Wireframe User Interface Software

Dalam mengerjakan suatu proyek website mungkin kalian akan dihadapkan dengan client yang ingin melihat gambaran kotor / sketsa mengenai werbsite yang akan kita buat. Pada kasus ini client mungkin saja dapat merubah sewaktu &#8211; waktu terhadap sketsa yang kita buat maka dari itu jika kita membuatnya langsung coding atau menggambar desain mungkin akan memakan waktu yang lama dalam pengerjaan-nya oleh karena itu kita butuh suatu software yang dapat memudahkan kita dalam pembuatan suatu sketsa dalam pembuatan website client itu.

## 6. Browser Add On

Tidak sedikit pekerjaan dapat cepat selesai dengan mengandalkan add on browser. Pada awalnya saya berpikir bahwa add on hanyalah fasilitas pelengkap atau aksesoris, tapi saya menyadari bahwa sekarang itu salah *lebay banget ya kata-katanya hahha. Ya tapi memang beneran lho bahwa dengan memanfaatkan add on pada browser kita, kita dapat dengan mudah menyelesaikan pekerjaan kita. Ga percaya? coba lihat deh add on berikut.

### FireBug

<a href="http://hitamcoklat.com/wp-content/uploads/2013/04/firebug1.3-img07.png" onclick="javascript:_gaq.push(['_trackEvent','outbound-article','http://hitamcoklat.com']);" rel="lightbox[783]" title="firebug1.3"><img class="aligncenter size-full wp-image-804" title="firebug1.3" src="http://hitamcoklat.com/wp-content/uploads/2013/04/firebug1.3-img07.png" alt="" width="414" height="344" /></a>

Firebug berfungsi / membantu kita dalam melihat source code dari sebuah website. Asyiknya lagi dengan add on ini kita dapat mengedit tampilan atau source code dari website yang kita akan edit / rubah secara live!. Dengan add on ini kita dapat meniru atau mengedit website yang akan kita jadikan referensi dalam membuat suatu website bahkan kita dapat mempelajari source bagaimana website itu berjalan (&#8220;tentunya source code yang dapat kita lihat hanya dari sisi client seperti Javascript/HTML/CSS)

### ColorZilla

<a href="http://hitamcoklat.com/wp-content/uploads/2013/04/ColorZilla.jpg" onclick="javascript:_gaq.push(['_trackEvent','outbound-article','http://hitamcoklat.com']);" rel="lightbox[783]" title="ColorZilla"><img class="aligncenter size-full wp-image-806" title="ColorZilla" src="http://hitamcoklat.com/wp-content/uploads/2013/04/ColorZilla.jpg" alt="" width="585" height="300" /></a>

Dengan menggunakan add on ColorZilla ini kita dapat mengetahui code dari warna yang ada dalam sebuah website.
